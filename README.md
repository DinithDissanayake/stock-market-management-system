# Stock Market Management System

A console-based Stock Market Management System developed in C as a Data Structures academic project.

## Features

- Add and manage stocks
- Register traders
- Buy and sell order management
- Order matching
- Transaction history
- Trade reporting
- Generate stock reports

## Data Structures Used

- Array
- Linked List
- Queue
- Stack

## Sorting Algorithms

- Bubble Sort
- Selection Sort
- Insertion Sort

## Technologies

- C Programming
- Data Structures
- Algorithms

## How to Run

Compile:

```bash
gcc Source_Code_Group-15.c -o stockmarket
```

Run:

```bash
./stockmarket
```
